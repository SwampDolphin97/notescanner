# CamScanner 

## Requirements

 - Python 2 or 3
 - NumPy 1.10 or later
 - SciPy
 - ImageMagick
 - Image module from PIL or Pillow

## Usage

```
./scanner.py IMAGE1 [IMAGE2 ...]
```

Building the examples (already in `example_output`):

```
make
```
